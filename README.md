<h1>Домашнее задание 1</h1>

<p>
  Вам нужно реализовать CLI приложения. Кроме кода, вам необходимо в качестве артефактов
  приложить текстовый файл с тем, как вы проверяли работоспособность вашего кода
  (просто копия команд и выводов из терминала).
</p>

<h2>1.1</h2>

<p>
  Написать упрощённый вариант утилиты <code>nl</code> — скрипт, который выдаёт в
  <code>stdout</code> пронумерованные строки из файла.
</p>

<p>
  Если файл не передан, то скрипт читает строки из <code>stdin</code>.
</p>

<p>
  Он должен работать так же, как:
</p>

<pre><code>nl -b a
</code></pre>

<h2>1.2</h2>

<p>
  Написать упрощённый вариант утилиты <code>tail</code> — скрипт, выводящий в
  <code>stdout</code> последние 10 строк каждого из переданных файлов.
</p>

<ul>
  <li>
    если передано больше одного файла, то перед обработкой очередного файла необходимо
    вывести его имя. Подробности смотрите в оригинальной утилите <code>tail</code>,
    ваш скрипт должен повторять форматирование;
  </li>
  <li>
    если не передано ни одного файла, то нужно вывести последние 17 строк из
    <code>stdin</code>.
  </li>
</ul>

<h2>1.3</h2>

<p>
  Написать скрипт, работающий так же, как утилита <code>wc</code>, вызванная без
  дополнительных опций.
</p>

<p>
  То есть для каждого переданного файла утилита выводит статистику (3 числа) и имя файла.
</p>

<p>При этом:</p>

<ul>
  <li>
    если передано больше одного файла, то в самом конце утилита выводит суммарную
    статистику (<code>total</code>);
  </li>
  <li>
    если ни одного файла не передано, то утилита считывает весь вход и печатает
    для него статистику без имени.
  </li>
</ul>
